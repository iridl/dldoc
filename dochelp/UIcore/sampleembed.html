Embed example

an example which uses iframe name to include a page with modifications

In this case, the iframe has a name portaldl. The uicore code copies this name to an attribute within of the body tag, so that css can be written using body[within='portaldl'] to change the appearance of the page when it is included in another page. In this case, the ui.css in /localconfig suppresses the dropdown navmenu and the first navitem in the control bar, which is always an uplink for our site.
