Finding Data

Are there ocean current data for the Atlantic Ocean in the Data Library?

The Data Library contains hindcast velocity data for the Pacific Ocean in a Climate Modeling Branch dataset, but I don't believe that the CMB currently produces such data for the Atlantic Ocean.

You might try contacting the CMB directly. You might also try contacting the people associated with the Climate Ocean Modeling Project at NASA's Jet Propulsion Laboratory. See the following website:

http://comp.jpl.nasa.gov/cgi-bin/wrap/yc/COMP

It appears that they are more concerned with ocean circulation variability on longer time scales, but they might know if the data you are looking for are available anywhere.

-- Michael Bell
