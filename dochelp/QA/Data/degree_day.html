Finding Data

Does the Data Library contain heating and cooling degree day information?

The Data Library has two datasets with heating and cooling degree data information.

These data variables are a bit hidden in their respective datasets, so let's look at each one in more detail.

  1. CPC Global Station Dataset

    Go to /SOURCES/.NOAA/.NCEP/.CPC/.GLOBAL/.monthly/.STATION/

    Select your desired station(s). Select the DATA>total>heating or cooling series of links under the Datasets and variables heading.

    The second one is a bit more obvious...

  2. NCDC First Order Summary of the Day Dataset

    Go to /SOURCES/.NOAA/.NCDC/.DAILY/.FSOD/.

    Select your desired station(s) then select either the cooling degree day or heating degree day link under the Datasets and variables heading.

You could have found this data by a keyword search here. Using "degree days" in your search should bring up both of the datasets we talked about here.

If these datasets do not contain data for the time period or location in which you are interested, you may find more complete data from CPC.
