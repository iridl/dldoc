Finding Data

Can you assist me in finding bioclimatic or land use data for Mozambique?

Most of the data sets in the Data Library are global or cover large regions. Very few are country-specific. However, you might still be able to find some useful data.

A good option for monthly station temperature and precipitation data is the National Climatic Data Center's (NCDC) Global Historical Climatology Network (GHCN) dataset. It is located here:

http://iridl.ldeo.columbia.edu/SOURCES/.NOAA/.NCDC/.GHCN/

"version 2" refers to the temperature data set and "beta version 2 (prcp)" is the precipitation dataset. It is possible to search by station name, ID number, lat/lon, etc., and to use the map to narrow the spatial domain as well.

Some gridded temperature and precipitation data can be found in the following locations:

http://iridl.ldeo.columbia.edu/SOURCES/.UEA/.CRU/
http://iridl.ldeo.columbia.edu/SOURCES/.NOAA/.NCEP/.CPC/.Merged_Analysis/.monthly/.v0107/
http://iridl.ldeo.columbia.edu/SOURCES/.NOAA/.NCEP/.CPC/.FEWS/

The "Merged Analysis" and FEWS datasets involve satellite estimations of rainfall, and access to the monthly UEA CRU NEW data values is restricted.

For soils, vegetation, land cover, etc., we have the NASA/ISLSCP Global Data Sets for Land-Atmosphere Models (GDSLAM) dataset located here:

http://ingrid.ldeo.columbia.edu/SOURCES/.NASA/.ISLSCP/.GDSLAM/

I think most of the applicable data are at 1 deg lat/lon resolution. Other vegetation/ecosystem data are located here:

http://ingrid.ldeo.columbia.edu/SOURCES/.ECOSYSTEMS/

The Holdridge Life Zones data are at 0.5 degree lat/lon resolution. Finally, you may already be aware of the U.S. Geological Survey (USGS)Africa Data Dissemination Service (ADDS). If not, see the following link:

http://edcsnw4.cr.usgs.gov/adds/

There are some cropland use intensity data for Mozambique available there.

You can also do a keyword search of the IRI Data Library in case I have left out some data sets that might be of interest to you.

--Michael Bell
