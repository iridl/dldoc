How do I overlay one piece of data onto another in the Data Viewer?

Let's overlay sea level pressure and wind vectors from CDAS-1

Open monthly Reanalysis for the example. (This example has already been limited in time and space. You can model this Ingrid text to create a similar set of data in which you are interested.)

If you were to go to the "Views" link to plot these data from here, you would only see the meridional wind data. You need to an additional line of Ingrid to overlay these data. Below the text that is already in the box, put the following line:

X Y fig: contours vectors :fig

You should now see all of the data displayed together like this. There is a link at the bottom of the page to go the Interactive Viewer if you would like to make changes to the image.
