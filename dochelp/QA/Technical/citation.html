How should I cite the data I am using from the Data Library?

While you are welcome to acknowlege the LDEO/IRI Data Library, it is not required.

p>

The proper documentation and references for the datasets within the Data Library can be found with the "dataset documentation" link on the dataset source page.

For example, if we are looking for the documentation of monthly Reynolds-Smith SST data, then we will click on the "dataset documentation" link to the right and near the bottom of the blue IRI banner on: http://iridl.ldeo.columbia.edu/SOURCES/.NOAA/.NCEP/.EMC/.CMB/.GLOBAL/.Reyn_SmithOIv1/.monthly/ . The specific location of this link will vary slightly between the source pages for the many datasets included in the Data Library, but it can always be found underneath the "Documents" heading near the top of the page. In addition, you will often find a journal or technical document reference for the dataset at the bottom of the page.
